PROJECT NAME: Smart Online University System (UoPeople-like LMS)

OBJECTIVE:
Build a scalable, production-ready online university platform similar to University of the People. The system must support Admins, Instructors, and Students, and include course management, video learning, assessments, results, certificates, and scholarship directory with intelligent automation.

TECH STACK:

* Backend: Laravel 12 (PHP 8.3+)
* Frontend: React (Vite + TypeScript)
* Styling: TailwindCSS
* State Management: Zustand or Redux Toolkit
* API: RESTful (JSON)
* Auth: Laravel Sanctum (token-based)
* Database: PostgreSQL (preferred) or MySQL
* Cache/Queue: Redis
* Storage: AWS S3 or Cloudflare R2
* Realtime (optional): Pusher / Laravel WebSockets

ARCHITECTURE:

* Modular Monolith (Domain-driven structure)
* Separate Modules: Auth, Users, Courses, Lessons, Assessments, Results, Notifications, Scholarships, AI
* API-first backend
* Frontend consumes API only (no Blade)

---

AGILE DEVELOPMENT PLAN:

WORK IN SPRINTS. DO NOT BUILD EVERYTHING AT ONCE.

Each sprint must include:

* Feature implementation
* API endpoints
* DB migrations
* Validation
* Basic UI (React)
* Testing stubs

* Protect routes via middleware
* Prevent XSS/SQL injection
